# Contributing to StadiumAI

We welcome contributions to StadiumAI. Please read and follow the guidelines below:

1. **Restructuring**: Always preserve the split between `backend/` and `frontend/` directories.
2. **Quality & Standard**: Follow high-performance, security, and accessibility standards.
3. **Tests**: Ensure that test coverage is kept above 80% on all code edits.
4. **Commits**: Ensure no API keys or environment secrets are committed.
